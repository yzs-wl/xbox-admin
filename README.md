## 📖 介绍

Xbox Admin 是一款企业信息化系统智能搭建工具，基于 Spring Boot + Vue 技术栈，帮助企业快速构建信息化管理系统。

---

## 📋 使用说明

1. **登录系统**  
   访问地址：[http://127.0.0.1:8080](http://127.0.0.1:8080)

2. **默认账号**  
   - 用户名：`admin`
   - 密码：`admin123`

3. **配置 AI 智能对话**  
   登录系统后，进入 **智能对话 → 模型配置** 菜单，选择对应模型并设置 API Key 即可使用大模型对话方式驱动系统各个模块，当然，也可以不用大模型，手工操作各模块菜单完成数据录入。  
   以 DeepSeek 大模型为例（性价比高），可在 [DeepSeek API Keys](https://platform.deepseek.com/api_keys) 页面注册申请 API Key。
