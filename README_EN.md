## 📖 Introduction

**Xbox Admin** is an enterprise information system intelligent builder based on **Spring Boot + Vue.js**. It is designed to help enterprises quickly build and manage information systems, providing full-cycle capabilities from application management and service governance to data model control.

### Key Features

- 🚀 **Quick Setup** — Ready to use out of the box, no complex configuration required
- 🤖 **AI-Powered** — Supports LLM-based intelligent dialogue to drive system modules via natural language
- 📦 **Full Lifecycle Management** — Covers the complete process including application registration, approval, online/offline, and category management
- 🔒 **Secure & Controllable** — Unified authentication and authorization, with comprehensive operation audit logging

## 🚀 Quick Start

### 1. Extract Installation Package

> Due to package size restrictions, the project has been split-compressed into **12 ZIP archives**. All must be extracted.

### 2. Launch the Service

Run `xbox-admin.exe` to start the backend service.

### 3. Access the System

| Item | Description |
|------|-------------|
| URL | [http://127.0.0.1:8080](http://127.0.0.1:8080) |
| Default Username | `admin` |
| Default Password | `admin123` |

---

## 🤖 AI Dialogue Configuration

After logging in, navigate to **Intelligent Dialogue → Model Configuration** in the menu, select the desired model and set the API Key to start driving system modules via LLM-powered conversations.

> You can also skip the LLM and manually operate each module to complete data entry.

### Recommended Model — DeepSeek

Offers excellent value for money. You can register and obtain an API Key at [DeepSeek API Keys](https://platform.deepseek.com/api_keys).
